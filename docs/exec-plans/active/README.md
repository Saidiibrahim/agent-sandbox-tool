# Active Plans

Active initiatives use one markdown plan plus JSON state. Feature implementation tracking lives in JSON, not in markdown task files.

Current state:

- No active initiatives are open in this repository.

Each active initiative directory must contain:

- `PLAN_<initiative>.md`
- `state/feature-list.json`
- `state/session-state.json`
- `state/progress.jsonl`

Create a new initiative directory here before starting any non-trivial change that needs durable handoff.
